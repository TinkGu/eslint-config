# eslint-config-react-ts-sonar

> eslint config with react, typescript and sonarjs

# Install

```bash
npm i eslint-config-react-ts-sonar -D
```

peer install, you can copy this on your `package.json`

```json
    "eslint": "^7.13.0",
    "typescript": "^3.8.3"
```

# Eslintrc

edit `.eslintrc.js`

```javascript
extends: ['react-ts-sonar'],
```
